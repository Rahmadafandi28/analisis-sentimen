\# Analisis Sentimen Ulasan Pengguna Gojek



Proyek ini bertujuan melakukan analisis sentimen ulasan aplikasi Gojek (positif atau negatif) menggunakan model BiLSTM.



\## Cara Menjalankan Program

1\. Install semua library:

pip install -r requirements.txt



2\. Jalankan training model:

python src/train.py



3\. Jalankan prediksi:

python src/predict.py





Hasil prediksi akan tersimpan dalam file CSV.



\## Teknologi yang digunakan

\- Python

\- TensorFlow

\- NLTK \& Sastrawi

\- Scikit-Learn

\- Pandas \& NumPy
